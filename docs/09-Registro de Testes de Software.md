## Relatório de testes de software

No dia 23 de novembro de 2025, foram realizados os primeiros testes para o registro de teste de software do Projeto Nexum, com objetivo de validar o desenvolvimento da aplicação e suas funcionalidades. Foram executados 12 casos de testes, a grande maioria com sucesso, apenas o CT-07 mostrou que a funcionalidade de exibir as filiais, não estava funcionanado como deveria. Após o ajuste do codigo, em um novo teste tivemos sucesso na funcionalidade.
